PROJECTS

Ringers

Django + React App

1 week, independent coding

Ringers is a full stack app for sourcing team players at the last minute. I used Python and Django REST Framework to serve my data from a Postgres database (visualised in TablePlus,) and React on the front end.  The app has multiple relationships and CRUD functionality. 

techTalk

MERN Stack App

1 week, group coding with 3 colleagues

Tech Talk is an app for arranging for tech related meetups. User can create events, see who is attending events and comment on events they are attending. We built the whole app in JavaScript. Our back-end used an Express API to serve the data from a Mongo database and is consumed by our front end built in React.

We used a combination of group coding using Visual Studio Live Share and pair programming to set up the framework, then we split to work on individual elements. Git and Trello were instrumental here for version control and ensuring our code wasn’t duplicated or conflicting. I focused on the forms - styling these with Skeleton, the language buttons - using Adobe Illustrator and Photoshop, and the map element using Mapbox API and react-map-gl.

Scran & Screen

Reacathon (React)

48 hours, coding in pairs

Scran and Screen allows users to search for restaurant recommendations near a chosen location, then shows movie times for nearby theatres. The app is built using React and uses 2 public RESTful API’s (Zomato and Cinelist) to retrieve the data. We coded collaboratively throughout the entirety of the project sharing a laptop and switching the driver and navigator every 30 minutes. This was the first project where I’d accessed an API and used Insomnia.

The Very Hungry Caterpillar

Vanilla JavaScript Game

1 week, independent coding

My first ever coding project was built with vanilla JavaScript using Flexbox and CSS3 to apply styling. I used DOM manipulation to enable players to guide the caterpillar to eat an array of foods that appear at random. Each bite increases the players score, and the length and speed of the caterpillar.

COURSE

Software Engineering Immersive, General Assembly, London

A full-time career accelerator designed to transform individuals into full-stack software engineers. Consisting of four modules: Programming Fundamentals and Computer Science; React; Node & Express; and Python & Django; the course is taught through expert-led lectures, discussions, interactive labs and homework assignments on a variety of languages, frameworks and libraries.

Progress was showcased through daily stand-ups, comprehensive portfolio projects and presentations with an emphasis on honing communication and collaboration skills, coding both in pairs and groups. Whilst the majority of material covered used the JavaScript and Python ecosystems, the curriculum was versatile enabling individuals to adapt to a variety of languages and technologies.
